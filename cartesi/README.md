# Cartesi SQLite Node

Cartesi rollup application with SQLite-based state persistence for L{CORE}.

Part of the [lcore-sdk](https://github.com/Modern-Society-Labs/lcore-sdk) ecosystem.

## Quick Start

```bash
npm install
npm run build
npm start
```

## Configuration

| Variable | Description | Default |
|----------|-------------|---------|
| `ROLLUP_HTTP_SERVER_URL` | Cartesi rollup server URL | `http://127.0.0.1:5004` |
| `AUTHORIZED_SENDERS` | Authorized addresses | Empty |
